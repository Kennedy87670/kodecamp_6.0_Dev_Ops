# Network Health Check

This is a beginner Bash script that checks basic network health and saves the result in a report file.

## Files

- `network_health_check.sh` - Bash script for the network checks
- `network_report.txt` - Report generated by the script

## What The Script Checks

1. Server information
   - Hostname
   - Current user
   - Date and time

2. Network information
   - IP address
   - Default gateway
   - DNS server

3. Internet connectivity
   - Pings `8.8.8.8`

4. DNS resolution
   - Resolves `google.com`

5. Website availability
   - Checks `google.com`
   - Checks `github.com`
   - Checks `amazon.com`

## How To Run

```bash
chmod +x network_health_check.sh
./network_health_check.sh
```

After running the script, open `network_report.txt` to see the saved report.
